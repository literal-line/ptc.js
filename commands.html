Welcome to PTC.js!

PTC.js is an in-development interpreter for the DSiWare app "Petit Computer." It is written in HTML5 for the web browser.

Key Features of PTC.js:
  • A 256 pixel by 384 pixel resolution, the DSi's native resolution.
  • 1:1 hand-drawn Petit Computer font (different from SmileBasic font).
  • Your mother (I couldn't think of anything else).
 

So far, there are very little commands, and they can only be inputted through the Javascript console. However, this project has only just started and is nowhere near completion.

Console Commands

The console layer is the highest layer on the upper screen. It used to manipulate strings of text. The following commands can be used with the console:

CLS

Parameters: none

JS format: runMode.console.cls()

PTC format: CLS

 

Clears the console screen.

LOCATE

Parameters: x position, y position

JS format: runMode.console.locate(x, y)

PTC format: LOCATE x,y

 

Sets position of console character placement.

COLOR

Parameters: palette number

JS format: runMode.console.color(palette number)

PTC format: COLOR palette number

 

Sets color of console character placement.

Graphic Commands

The graphic layer is the lowest layer on the upper screen. It is used to lines and shapes. The following commands can be used with the graphic layer.

NOTE: graphic commands are supposed to have their own color palette, but this has not been implemented yet.

GCLS

Parameters: none

JS format: runMode.graphic.gcls()

PTC format: GCLS

 

Clears the graphic screen.

GLINE

Parameters: x start, y start, x end, y end, palette number

JS format: runMode.graphic.gline(x1, y1, x2, y2, palette number)

PTC format: GLINE x1, y1, x2, y2, palette number

 

Draws a line on the graphic screen.
